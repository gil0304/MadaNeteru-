# カスタマイズガイド（文言・キャラ配置の変え方）

このアプリの **画面の文言・キャラクター・色** を変えるための手引きです。
Xcode でファイルを開き、該当行を書き換えて再ビルドするだけで反映されます。
（`MadaNeteru?/` 配下の `.swift` は自動でビルド対象になるので、プロジェクト設定の編集は不要です。）

---

## 1. キャラクターの入れ替え

### 1-1. どの画面に誰がいるか（現在の割り当て）

各画面の `CharacterView(character: .○○)` の `.○○` を変えるだけで担当キャラが変わります。

| 画面 | キャラ | ファイル:行 |
|---|---|---|
| ログイン | `.emily` | [OnboardingView.swift:43](../MadaNeteru?/Views/Onboarding/OnboardingView.swift) |
| 許可 | `.gil` | [OnboardingView.swift:99](../MadaNeteru?/Views/Onboarding/OnboardingView.swift) |
| ホーム（左） | `.kamimu` | [HomeView.swift:82](../MadaNeteru?/Views/Home/HomeView.swift) |
| ホーム（右） | `.honopi` | [HomeView.swift:83](../MadaNeteru?/Views/Home/HomeView.swift) |
| 予定 | `.watami` | [EventsView.swift:82](../MadaNeteru?/Views/Events/EventsView.swift) |
| ルール | `.shiro` | [RulesView.swift:122](../MadaNeteru?/Views/Rules/RulesView.swift) |
| 予定詳細 | `.aiueo` | [EventDetailView.swift:109](../MadaNeteru?/Views/Events/EventDetailView.swift) |
| デフォルト設定 | `.yucha` | [DefaultSettingsView.swift:94](../MadaNeteru?/Views/Settings/DefaultSettingsView.swift) |
| 鳴動（充電） | `.yucha` | [ChargeAlarmView.swift:19](../MadaNeteru?/Views/Alarm/ChargeAlarmView.swift) |
| 鳴動（起床） | `.emily` | [ChargeAlarmView.swift:19](../MadaNeteru?/Views/Alarm/ChargeAlarmView.swift) |

**使えるキャラ名（`.○○`）**：`.yucha` `.aiueo` `.shiro` `.watami` `.gil` `.emily` `.kamimu` `.honopi`

例：ログインを `.emily` から `.shiro` に変える →
```swift
CharacterView(character: .shiro, height: 264)   // OnboardingView.swift:43
```

### 1-2. キャラのサイズ

`height:` の数値がキャラの高さ（pt）です。大きくしたいときはここを増やします。
`onDark: true` は暗い背景（ホームのヒーロー・鳴動画面）用のプレースホルダ色指定です。

### 1-3. 鳴動画面のキャラは1か所で決まる

充電・起床の鳴動キャラは [ChargeAlarmView.swift:19](../MadaNeteru?/Views/Alarm/ChargeAlarmView.swift) の1行で決まります。
```swift
var character: AppCharacter { self == .charge ? .yucha : .emily }
//                                    充電なら↑    起床なら↑
```

## 2. キャラ画像の差し替え・追加

### 2-1. 既存キャラの画像を差し替える
`MadaNeteru?/Assets.xcassets/char-○○.imageset/` の中の PNG を置き換えます。
- **重要**：`UIImage(named:)` が探すのは **imageset フォルダ名**（`char-○○`）です。
- フォルダ内の実際の PNG ファイル名は `Contents.json` が参照します。ファイル名を変えたら `Contents.json` の `"filename"` も合わせてください。
- 画像は**透過PNGをそのまま**入れればOK（透過処理などは不要）。水彩タッチで一部が半透明でも、表示側で白い背景プレートを敷いて透けないようにしています（[DesignKit.swift の CharacterView](../MadaNeteru?/Views/Components/DesignKit.swift)）。

### 2-2. 新しいキャラを足す
1. `Assets.xcassets` に `char-新名.imageset` を追加（PNGを入れ、`Contents.json` を用意）。
2. [DesignKit.swift](../MadaNeteru?/Views/Components/DesignKit.swift) の `enum AppCharacter` に case を1つ足す：
   ```swift
   enum AppCharacter: String {
       case yucha, aiueo, shiro, watami, gil, emily, kamimu, honopi, 新名, undefined
       // ...
       var displayName: String {
           switch self {
           // ...
           case .新名: return "表示名"
           }
       }
   }
   ```
   `assetName` は `"char-\(rawValue)"` なので、**case 名とフォルダ名（char-以下）を一致**させてください。
3. 使いたい画面で `CharacterView(character: .新名, height: ...)`。

## 4. 充電確認アラームの「くり返し回数・間隔」

充電するまで何回・何分おきに鳴らすかは [AlarmPlanner.swift](../MadaNeteru?/Services/AlarmPlanner.swift) の定数で変えられます。

```swift
static let chargeRepeatIntervalMinutes = 15   // 再通知の間隔（分）
static let chargeRepeatMaxCount = 4           // 初回含む総回数（0/15/30/45分 → 最長45分）
```

- 例：「充電するまで最長1時間」にしたい → `chargeRepeatMaxCount = 5`（0/15/30/45/60分）。
- 充電を検知すると残りは自動キャンセルされます（[AppModel.swift](../MadaNeteru?/Stores/AppModel.swift) の `autoStopChargeIfCharged` / `cancelTonightChargeAlarm`）。
- AlarmKit のアラーム数を抑えるため、チェーン化するのは**直近（今夜）の1晩のみ**。先の夜は単発で、その夜が近づいて再構築が走ったときに展開されます。

## 5. 色・テーマの変え方

配色は [Support/Theme.swift](../MadaNeteru?/Support/Theme.swift) に集約されています。

- **アクセントのオレンジ**：`Theme.orange`（`#EE5A24`）。ボタン・選択中の曜日サークル・強調文字などに使用。
- 種別ごとのアイコン色は `Theme.color(for: AlarmType)`。
- 背景・区切り線・文字色（`groupedBg` / `separator` / `label` / `secondary` など）もここ。

1か所変えると全画面に反映されます。特定画面だけ変えたい場合は、その画面の `foregroundStyle(...)` / `background(...)` を個別に指定してください。

## 6. 共通UI部品の在りか

再利用パーツは [Views/Components/DesignKit.swift](../MadaNeteru?/Views/Components/DesignKit.swift) にまとまっています。

| 部品 | 役割 |
|---|---|
| `CharacterView` | キャラ画像表示（サイズ・白背景プレート・プレースホルダ） |
| `SpeechBubble` | 吹き出し（しっぽの向き `tail:`、背景色 `bg:`） |
| `DayCircle` | 曜日サークル（サイズ `size:`、選択色） |
| `EmojiIcon` / `AlarmTypeIcon` | 角丸＋絵文字のアイコン |
| `SourceBadge` | 個別/曜日/デフォルトの色バッジ |
| `PrimaryButton` | オレンジの角丸ボタン |

見た目のトーンを揃えたいときは、まずここの部品を直すと全画面に効きます。

## 7. 変更後の確認

```sh
# シミュレータ（iOS 26）にビルド
xcodebuild -project "MadaNeteru?.xcodeproj" -scheme "MadaNeteru?" \
  -destination 'generic/platform=iOS Simulator' -configuration Debug build

# 開発用デモ起動（ログイン・許可を飛ばしてサンプルデータで確認、DEBUG限定）
xcrun simctl launch <SIM_UDID> app.Ochiai.gil.MadaNeteru- -demoMode
#   -demoTab 0|1|2                 ホーム/予定/ルール で起動
#   -demoScreen detail|weekday|default|charge|wake   個別画面をプレビュー
#   -onboardStep 1                 許可画面から起動
```

Xcode の `Run` でも確認できます。文言・キャラの変更はビルドし直せば即反映されます。
