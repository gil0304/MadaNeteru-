# まだねてる？ 使い方ガイド

Googleカレンダーと連動して、**「明日の予定に遅れない状態を、前日の夜に自動で整える」** アラームアプリです。
予定を登録するアプリではなく、すでにある予定に合わせて **起床・前日確認・充電確認** のアラームを自動で用意します。

---

## 1. 初回セットアップ

### ① ログイン
1. アプリを開くと **ログイン画面**（キャラが挨拶）が出ます。
2. 「**Google でログイン**」をタップ。
3. カレンダーの予定を**読み取り専用**で取得します（予定の書き換えはしません）。

### ② 許可（4つ）
ログイン後、アラームを確実に鳴らすために4つの許可を求めます。

| 許可 | 何のため |
|---|---|
| 📅 カレンダー | 明日の予定を読み取る |
| ⏰ アラーム（AlarmKit） | 長く・強く鳴る重要アラーム |
| 🔋 バッテリー | 充電済みかどうかの確認 |
| 🔔 通知 | 軽いリマインド |

「**すべて許可してつづける**」で完了。以降はホーム画面から始まります。

---

## 2. 画面の構成（3タブ）

下のタブバーで **ホーム / 予定 / ルール** を切り替えます。

### 🏠 ホーム
- 今夜〜明朝の状況をキャラが要約（例：「明日は予定が3件！ 10:00の面接にあわせて、07:00に起こすね。」）。
- 「**今夜のチェック**」に、今夜やる起床アラーム・充電確認が並びます。
- 行をタップすると、**アラームが鳴ったときの画面（鳴動画面）をその場で確認**できます。

### 📅 予定
- 上のセグメントで **明日 / 今日 / 7日間** を切り替え。
- 各予定に、適用中の起床時刻や 🔋（充電確認）バッジが表示されます。
- 予定をタップすると **予定詳細**（この予定だけの設定）へ。

### ⏰ ルール
- いま選んでいる予定に **どのルールが適用中か**を、色バッジ（下記）で表示。
- ここから **曜日ごとのルール** と **デフォルト設定** に入れます。

---

## 3. アラームの決まり方（優先順位）

同じ種類のアラーム（起床・前日確認・充電確認など）は、次の順で **上が勝ち** ます。

```
① 予定ごと（個別）  ＞  ② 曜日ごと  ＞  ③ 全体デフォルト
```

- 色バッジ：**個別＝赤 / 曜日＝オレンジ / デフォルト＝グレー**。
- 例：普段は「デフォルト起床 07:00」でも、水曜だけ「曜日ルールで 06:30」にすれば水曜は 06:30。さらにその日の面接だけ「個別で 06:00」にすればその予定は 06:00。

### 設定する場所

| やりたいこと | 場所 |
|---|---|
| 特定の予定だけ時刻を変える | 予定 → 予定詳細 → 時刻をタップ |
| 特定の予定だけアラームを止める | 予定詳細で行を**左スワイプ →「オフ」**、または「この予定はアラーム不要にする」 |
| 曜日ごとに決める | ルール → 曜日ごとのルール |
| 全体の基本を決める | ルール → デフォルト設定 |

---

## 4. 各設定画面

### 予定詳細
- 「この予定のアラーム」で **起床 / 前日確認 / 充電確認 / 出発** の時刻を個別調整。
- **時刻をタップ**で変更、変更後は一覧にすぐ反映されます。
- **左スワイプ**でその種類だけオフ。
- 「**この予定はアラーム不要にする**」でこの予定を丸ごと対象外に。

### 曜日ルール
- 上の**曜日サークル**（月〜日）で曜日を選択。
- その曜日の 起床 / 前日確認 / 充電確認 / 出発 を設定。
- 「**この曜日を有効**」でその曜日のルールをまとめてオン/オフ。

### デフォルト設定
- **デフォルト起床 / 前日の確認 / 充電確認 / 未設定チェック** の基準時刻。
- **スヌーズ間隔・アラーム音・通知音**。
- **アカウント**（ログイン中のGoogle・各種許可の状態・ログアウト）。

---

## 5. 充電確認について

夜、スマホの充電を忘れて朝アラームが鳴らない…を防ぐ機能です。

- 設定時刻になったら「**充電した？**」と確認します。
- **充電済み（充電中/満充電）と確認できた場合は鳴りません。**
- 充電状態が分からない・未確認のときは、安全側に倒して**鳴らします**。
- **充電するまで、15分おきに最大4回（＝最長45分）くり返し鳴ります。**
- ホームや予定を開いたとき・バックグラウンド更新時に充電を検知すると、その夜の残りの充電アラームは自動で止まります。「充電した」を押しても止まります。

> ⚠️ iOSの制約上、「充電中なら絶対に鳴らさない」を100%保証はできません。アプリが発火時刻より前に一度も動けなかった場合（＝下記のような状況）は、安全のため鳴ることがあります。
>
> **アプリがバックグラウンドで動けない主なケース**：①アプリをスワイプで完全終了している ②低電力モードがオン ③「Appのバックグラウンド更新」がオフ ④OSが「今は動かさない」と判断（低頻度アプリ）⑤電池残量が少ない/発熱/通信不可 ⑥発火直前に背景実行の枠が回ってこなかった。

---

## 6. アラームが鳴ったときの操作

### 起床アラーム
- 「**止める**」／「**スヌーズ**」。

### 充電確認アラーム
- 「**充電した**」：充電を確認して停止（充電できていない場合は「まだ確認できません」と表示）。
- 「**15分後に再通知**」：15分後にもう一度鳴らします。
- 「**今日は不要**」：今夜はもう鳴らさない。

> ホーム画面の「今夜のチェック」の行をタップすると、この鳴動画面を**事前に確認**できます。

---

## 7. こんなときは

- **明日予定がない** → ホームで「明日は予定がないよ。ゆっくり休んでね。」と表示され、起床アラームは用意されません（デフォルト設定で常時の起床を入れている場合を除く）。
- **予定を変えたのに反映されない** → 予定は次回のカレンダー同期（アプリを開く/バックグラウンド更新）で追従します。
- **ログアウトしたい** → ルール → デフォルト設定 → アカウント → ログアウト。
